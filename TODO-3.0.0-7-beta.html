New Workload Installers

Previously, some users experienced problems with large downloads. Separating the Inno Installers into logical groups will ease some of this burden. It also makes for easier maintenance going forward.

Installer Groups

NOTE - This method only delivers the binaries. In cases such as Qt, VS Code, Postgres, etc, additional installation action will be required.
